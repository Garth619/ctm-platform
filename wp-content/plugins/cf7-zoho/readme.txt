Send Contact Form 7, Contact Form Entries Plugin and many other contact form submissions to zoho CRM.

== Description ==

Contact Form 7 Zoho Plugin sends form submissions from [Contact Form 7](https://wordpress.org/plugins/contact-form-7/), [Contact Forms Entries  Plugin](https://wordpress.org/plugins/contact-form-entries/) and many other popular contact form plugins to Zoho CRM. Learn more at [crmperks.com](https://www.crmperks.com/plugins/contact-form-plugins/contact-form-zoho-plugin/?utm_source=wordpress&utm_medium=directory&utm_campaign=zoho_readme)


== How to Setup ==

* Go to "Zoho Accounts" tab and add new account.
* Go to "Zoho Feeds" tab , create new feed.
* Map required Zoho fields to contact form 7 fields.
* Send your test entry to Zoho CRM.
* Go to "Zoho Logs" tab and verify, if entry was sent to Zoho CRM.


**Connect Zoho CRM account**

Connect any contact form 7 to Zoho account by simply entering zoho API key. Easily connect multiple Zoho account to wordpress.

**Map Zoho CRM fields**

Map any contact form 7 fields to any Zoho object(Contacts, Account, Lead, Order, Case) fields. No limitation on number of fields.

**Filter contact form 7  submissions**

Send all or filter contact form 7 submissions sent to Zoho CRM based on user input.

**Manually send to Zoho CRM**

Send contact form 7 submissions to Zoho CRM on form submission. You can manually send contact form 7 submissions to Zoho CRM.

**Zoho logs**

View a detailed log of each contact form 7 submission sent (or not sent) to Zoho CRM and easily resend contact form 7 submission to Zoho CRM.

**Send Data As Zoho object Notes**

Send one to many contact form 7 fields as Zoho object(Contacts, Account, Lead, Order, Case) notes.


**Create Or Update Contact in Zoho (Premium feature)**

If an entry(Contacts, Account, Lead, Order, Case) already exists in Zoho CRM, update it otherwise create a new entry in Zoho CRM.
